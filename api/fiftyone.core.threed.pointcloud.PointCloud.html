fiftyone.core.threed.pointcloud.PointCloud

class documentation

class PointCloud(Object3D): (source)

Constructor: PointCloud(name, pcd_path, material, center_geometry, ...)

View In Hierarchy

Represents a point cloud.

Parameters
namethe name of the point cloud
pcd_paththe path to the .pcd file. The path may be either absolute or relative to the directory containing the .fo3d file
materialthe material of the point cloud. If not specified, defaults to a new instance of fiftyone.core.threed.PointCloudMaterial with its default parameters
center_geometrywhether to center the geometry of the point cloud. Defaults to False
flag_for_projectionwhether to flag the point cloud for usage in orthographic projection. Each fiftyone.core.threed.Scene can have at most one asset flagged for orthographic projection. Defaults to False. If multiple assets are flagged, the first one will be chosen
visibledefault visibility of the point cloud in the scene
positionthe position of the object in point cloud space
quaternionthe quaternion of the point cloud in object space
scalethe scale of the point cloud in object space
Raises
ValueErrorif pcd_path does not end with .pcd
